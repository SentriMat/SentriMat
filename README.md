# SentriMat: IoT-Enabled Package Security System

**SentriMat** is a privacy-first, camera-free security solution designed to mitigate porch piracy through **Dynamic Force-Signature Analysis**. By utilizing a high-frequency sensor matrix, the system distinguishes between authorized deliveries, environmental noise, and unauthorized theft.

* **Impact Force Fingerprinting:** Samples contact force during the first 500ms to identify package material.
* **Spatial CoP Tracking:** Monitors for package tilting or sliding using a multi-channel sensor matrix.
* **Privacy-First Design:** Full security monitoring without intrusive cameras.
* **Automated Data Logging:** Generates time-stamped CSV files for detailed trend analysis.
* **Dual-Factor Verification:** Requires a physical footprint signature and a digital handshake.
